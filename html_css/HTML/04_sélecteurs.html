Les balises HTML

Les sélecteurs de type ciblent le éléments de la page HTML par les balises. Par exemple, si nous voulons cibler tous nos titres de niveau <h4> de la page HTML nous utiliserons le sélecteur de type h4

HTML :
<h4>Bonjour à toujours</h4>
<h4>Au revoir à tous</h4>

CSS :
h4>{
...}

Les sélecteurs de classe permettent de cibler un élément en fonction de la valeur de l'attribut class attaché à cet élément. Les sélecteurs de class permettent davantage de précision car ils permettent de cibler les éléments d'un groupe ayant une class identique plutôt que toutes les balises html de la page. Les sélecteurs de class permettent d'appliquer la même mise en forme en attribuant la même valeur dans l'attribut class à différentes balises HTML. En CSS les class sont appelées précédées d'un point. Ici le sélecteur de class ciblera tout élément contenant la class dont la valeur est "zone 1" , y inclus les éléments division et paragraphe.

Les sélecteurs d'IDsont encore plus précis que les sélecteurs de class puisqu'ils ciblent un élément unique. Tout comme les sélécteurs de class utilisent la valeur d'une class présente dans le html, les sélecteurs d'ID utlisent la valeur de l'attribut ID.

/!/ ATTENTION ! Très important: peu importe la balise dans laquelle ils sont insérés, la valeur des attributs ID ne peuvent être utilisés qu'une seule fois dans un fichier html. C'est pourquoi les ID sont réservés pour les éléments significatifs.
En css les ID sont appelées précédés d'un dièse (ou hash tag). Ici le sélecteur d'ID ciblera uniquement l'élément HTML qui possède un attribut ID dont la valeur est "zone3".
