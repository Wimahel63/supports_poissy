Les boucles

La boucle while

La boucle while va nous permettre de répéter une série d'actions tant qu'une condition est vraie.

la boucle do...while

La boucle for

Elle nous permet de répéter une série d'instructions à partir d'une valeur donnée jusqu'à une certaine valeur seuil.

Les Fonctions
