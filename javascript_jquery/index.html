rappel:

FRONT-END

html: langage de balises qui permet d'afficher les informations

css: langage de mise en forme. Permet de styliser nos éléments. --> bootstrap: FRAMEWORK CSS! (framework = librairie)

javascript: langage permettant de rendre les pages interactives en y ajoutant des mouvements, des évènements, de l'animation. --> jquery: librairie basée sur le javascript. (write less, do more). Cette partie , le front-end, est celle visible par l'utilisateur.

BACK-END

PHP (Hypertext Preprocessor) : langage de programmation qui permet de produire des pages dynamiques et de développer des fonctionnalités. (ex:entrer un mdp, ajouter bun article au panier...).

SQL (Structured query language): langage de requête permettant d'échanger des informations avec une base de données (CRUD, Create, Read, Update, Delete)

----prenons l'exemple d'un forum:

--> PHP pour rendre le site fonctionnel

-->SQL pour sauvegarder mes messages, utilisatuers etc.

-->JS pour mettre des smileys dans les messages etc.

-->HTML pour le retour visuel.

-->CSS pour la mise en forme

JS :langage qui permet de rendre nos pages intercatives. Integration de notre script soit directement dans le HTML, soit dans un fichier .js que nous allons lier (linker, comme le stylesheet .css). DOM: Document Object Model. Representation (Object) de ma page web, qui permet d'ajouter et/ou modifier le contenu, ou le style de mon document (ma page web).
